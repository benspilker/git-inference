The standard context window for a free, unsigned-in ChatGPT session is typically around 4096 tokens. This is a combined total of both the user's messages and the assistant's responses.

Here’s what that means for your conversation:

As you send more messages and the assistant responds, the context window will "fill up."

Each message and response (including any system message or prior chat history) contributes to this token count.

If the token limit is reached, the assistant will lose earlier parts of the conversation, which can cause the responses to become less accurate or coherent as context degrades.

In general, depending on the length of your messages and the assistant’s replies, you can expect roughly 7-10 messages (for both sides combined) before the context starts to degrade, but this varies depending on the length and complexity of the content.
