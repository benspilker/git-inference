GitHub Actions servers are primarily located in the United States, but GitHub uses multiple data centers around the world to host its infrastructure, including servers in North America, Europe, and other regions. These data centers ensure high availability and performance across different geographic areas.

In particular, GitHub's cloud services are hosted on Microsoft Azure infrastructure, which spans a global network of data centers. So while a large portion of the operations may be in the U.S., there is likely some level of distributed infrastructure in other regions for redundancy and performance optimization.

If you have concerns about data residency, GitHub provides options and configurations for specific regions in some cases, especially for enterprise users.
