The first 64-bit PC processor for the consumer market was the AMD Athlon 64, released in September 2003. It was the first widely available desktop CPU to support a full 64-bit instruction set while remaining compatible with 32-bit software.

If you’re asking specifically about the very first 64-bit x86 PC, AMD’s Athlon 64 is generally credited, though 64-bit workstations existed earlier (like those using DEC Alpha or Itanium CPUs) but were not typical “PCs” in the consumer sense.
