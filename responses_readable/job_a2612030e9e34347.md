Here’s a Python script that uses the Pillow library to create simple image renderings of fruits and vegetables. For more advanced rendering (like 3D models or realistic images), you'd typically need something more powerful, such as a rendering engine or a pre-trained AI model. But for basic image rendering, Pillow can work for creating simple 2D images, like representations of fruits and vegetables.

First, you will need to install Pillow if you haven't already:

Bash
pip install Pillow

Now, let's create a Python script that renders images of basic fruits and vegetables like apples, bananas, and carrots.

Python
Run
from PIL import Image, ImageDraw

def create_apple_image():
    # Create a blank canvas
    img = Image.new('RGB', (200, 200), color='white')
    draw = ImageDraw.Draw(img)

    # Draw an apple (circle with a stem)
    draw.ellipse([50, 50, 150, 150], fill='red', outline='black')  # Apple body
    draw.line([100, 50, 100, 30], fill='brown', width=5)  # Stem

    # Save the image
    img.save("apple.png")
    img.show()

def create_banana_image():
    # Create a blank canvas
    img = Image.new('RGB', (200, 200), color='white')
    draw = ImageDraw.Draw(img)

    # Draw a banana (a curved yellow shape)
    draw.arc([50, 50, 150, 150], start=0, end=180, fill='yellow', width=15)

    # Save the image
    img.save("banana.png")
    img.show()

def create_carrot_image():
    # Create a blank canvas
    img = Image.new('RGB', (200, 200), color='white')
    draw = ImageDraw.Draw(img)

    # Draw a carrot (triangle shape)
    draw.polygon([(100, 50), (150, 150), (50, 150)], fill='orange', outline='black')  # Carrot body
    draw.line([100, 50, 100, 30], fill='green', width=5)  # Carrot top

    # Save the image
    img.save("carrot.png")
    img.show()

if __name__ == "__main__":
    create_apple_image()
    create_banana_image()
    create_carrot_image()
How this script works:

Pillow Library: We use Pillow to generate the images and draw basic shapes.

Apple: We create a red circle with a brown line to represent a stem.

Banana: We use the arc method to create a yellow banana-like curved shape.

Carrot: We draw a triangle for the body and a line for the green carrot top.

Running the Script:

The script will create and save three images: apple.png, banana.png, and carrot.png in the current working directory.

The images will also be displayed on the screen when the script is run.

Feel free to extend this for more fruits and vegetables or make the shapes more sophisticated.
