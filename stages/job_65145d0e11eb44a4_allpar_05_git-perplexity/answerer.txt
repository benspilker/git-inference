Ping is a network utility used to test reachability and measure round-trip time to a host. It typically sends ICMP echo requests and reports replies, latency, and packet loss.

Answer

A basic ping tests whether a host is reachable and how quickly it responds.

It shows metrics like round-trip time (latency) and packet loss for the test window.

Key details

How it works: sends ICMP echo requests to a target, waits for echo replies, and summarizes results.
wikipedia

Common output: number of packets transmitted, received, loss percentage, and min/avg/max latency.
wikipedia

Typical uses: quick connectivity checks, diagnosing network issues, and assessing path responsiveness.
